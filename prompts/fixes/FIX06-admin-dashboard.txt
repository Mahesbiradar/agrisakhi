In users/views.py add admin-only views:
Use permission class: IsAuthenticated + request.user.role == 'admin'

Create IsAdmin permission class:
class IsAdmin(BasePermission):
    def has_permission(self, request, view):
        return request.user.is_authenticated and request.user.role == 'admin'

AdminStatsView(APIView) — permission: IsAdmin
  GET: return {
    total_users: User.objects.count(),
    farmers: User.objects.filter(role='farmer').count(),
    labours: User.objects.filter(role='labour').count(),
    providers: User.objects.filter(role='provider').count(),
    total_jobs: Job.objects.count(),

    pending_applications: Application.objects.filter(status='pending').count(),
    total_services: ServiceListing.objects.count(),
  }

AdminUsersView(generics.ListAPIView) — permission: IsAdmin
  Returns all users. Query param ?role= to filter. ?search= to search by name/phone.
  Serializer includes: id, name, phone, role, village, district, is_active, created_at

AdminUserToggleView(APIView) — permission: IsAdmin
  POST with {user_id}: toggle user.is_active
  Returns updated is_active status

AdminPasswordResetView(APIView) — permission: IsAdmin
  POST with {user_id, new_password}
  Resets password directly

AdminJobsView(generics.ListAPIView) — permission: IsAdmin
  Returns all jobs with farmer name. ?status= filter.

AdminJobCloseView(APIView) — permission: IsAdmin
  POST with {job_id}: sets job.status='closed'

AdminServicesView(generics.ListAPIView) — permission: IsAdmin
  Returns all service listings
