STEP 2 — Create src/lib/api.js
Axios instance with baseURL from import.meta.env.VITE_API_URL
Request interceptor: attach Authorization: Bearer from localStorage key 'agrisakhi_access'
Response interceptor: on 401, attempt refresh via POST /auth/token/refresh/ with 'agrisakhi_refresh'
  If refresh ok: save new access token, retry request
  If refresh fails: clear both tokens from localStorage, redirect to /login

Export named API function groups:

  usersAPI: { getProfile(), updateProfile(data), getNearby(params) }

STEP 3 — Create src/store/authStore.js
